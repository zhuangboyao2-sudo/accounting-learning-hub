## 兩種截然不同的計算邏輯

六大計費項目中，「獎金」與其餘五項（兼職薪資、執行業務收入、股利、利息、租金）的計算邏輯**並不相同**，這是實務上最容易混淆的地方：

**股利、利息、租金、執行業務收入、兼職薪資所得**：採「達標即整筆課徵」——單筆給付金額達新臺幣<TaxParam path="secondGenNhiSupplementaryPremium.itemThresholdNTD" />元（含）以上，就**整筆給付金額**（不是只有超過部分）乘以<TaxParam path="secondGenNhiSupplementaryPremium.rate" format="percent" />計算；未達門檻則完全免扣。

**全年累計超過當月投保金額4倍部分之獎金**：採「僅就超過部分課徵」——只有超過當月投保金額4倍的**部分**才計算補充保費，沒有下限門檻（哪怕只超過1元也要就超過部分計算），4倍以內的獎金完全不計。

## 計算範例一：股利所得（整筆課徵）

某股東領取單筆現金股利25,000元，已達2萬元門檻：

```
補充保費 = 25,000（全額，非扣除2萬元後的餘額）× 2.11% = 527.5元 → 依規定四捨五入至元
```

若單筆股利為19,000元，未達2萬元門檻，則完全免扣補充保費（不是扣「0元」還要處理，是整筆都不觸發扣取義務）。這與所得稅累進課稅「只對超過部分課稅」的直覺完全不同，是初學者最容易誤解之處。

## 計算範例二：獎金（超額課徵）

某員工月投保金額6萬元，全年領取獎金（如年終獎金）30萬元。當月投保金額4倍＝6萬×4＝24萬元：

```
超過4倍部分 = 300,000 － 240,000 = 60,000元
補充保費 = 60,000 × 2.11% = 1,266元
```

只有超過24萬元的部分（6萬元）才計算補充保費，前面24萬元完全不扣——這與股利、利息等「整筆課徵」的邏輯明顯不同。

## 給付金額上限

無論哪一類所得，單筆給付金額超過新臺幣<TaxParam path="secondGenNhiSupplementaryPremium.singlePaymentCapNTD" format="number" />元的部分，不再計收補充保費——也就是即使股利、租金等單筆金額再高，補充保費的計算基礎都以1,000萬元為上限封頂。

## 執行業務收入與所得稅扣繳的併行處理

工廠支付會計師報酬3萬元（單筆達2萬元門檻），須同時處理：

```
所得稅執行業務報酬扣繳 = 30,000 × 10% = 3,000元
二代健保補充保費 = 30,000 × 2.11% = 633元
```

會計師實際收到之淨額 = 30,000 － 3,000（所得稅扣繳）－ 633（補充保費）= 26,367元。扣費義務人須分別計算兩筆扣款、分別開立憑證與申報，兩者互不影響、互不抵充。

## 常見錯誤

- 誤以為股利、利息、租金等所得也是「只對超過2萬元的部分」課徵補充保費，忽略這五類所得是達門檻後就整筆金額課徵，計算基礎完全不同於獎金項目
- 誤以為給付會計師報酬只需處理所得稅扣繳，忽略若單筆金額達2萬元門檻，還須另外計算並扣取二代健保補充保費
- 忘記單筆給付超過1,000萬元的部分不再計收補充保費，誤以為金額越高補充保費就無上限持續累加
