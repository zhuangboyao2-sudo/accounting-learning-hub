## 紙本發票遺失的處理方式

工廠若遇到紙本發票遺失，處理方式依遺失的聯別而不同：

```
一、遺失空白發票：應即日敘明遺失之發票種類、字軌號碼，向主管稽徵機關申報核銷
二、遺失存根聯：取得買受人蓋章證明之原收執聯影本代替
三、遺失扣抵聯或收執聯：取得原銷售營業人蓋章證明之存根聯影本，或以未遺失聯之影本自行蓋章證明，得作為進項稅額扣抵憑證或記帳憑證
```

## 雲端發票是什麼

雲端發票是買方使用財政部核准載具（手機條碼、悠遊卡、icash卡、會員卡）儲存、且未列印紙本證明聯的電子發票。營業人與非營業人交易時，應主動告知買方可用哪些載具索取雲端發票，並提示查詢、對獎、捐贈相關資訊；買方持共通性載具索取時，營業人不得拒絕。

## 雲端發票專屬獎：對消費者的加碼誘因

除了統一發票原本的特別獎、特獎、頭獎等獎項外，財政部另外設有「雲端發票專屬獎」，只有使用載具儲存雲端發票的買方才有資格參加，獎項與現行組數如下（財政部依統一發票給獎辦法規定的組數上限逐期公告實際開獎組數）：

```
百萬元獎：100萬元，現行約30組
千元獎：2,000元，現行約1萬6千組
百元獎：800元，現行約10萬組
五百元獎：500元，現行約315萬組
```

雲端發票專屬獎與紙本統一發票獎項（特別獎、特獎、頭獎至六獎）同一天由財政部開獎，工廠若面向消費者銷售（例如工廠直營門市），可將此作為鼓勵客戶使用雲端發票、減少紙本作業的說明重點。

## 中獎通知與領獎方式

若買方使用手機條碼等已完成身分認證的載具，中獎後由電子發票整合服務平台主動以電子郵件通知，也可至平台或超商多媒體服務機查詢。領獎方式分兩種：

```
一、已於整合服務平台或載具APP設定匯款帳戶：中獎獎金於開獎日次月自動匯入，且免依印花稅法規定被扣繳4%印花稅
二、未設定匯款帳戶：需至超商多媒體服務機（如ibon、FamiPort）列印中獎發票證明聯，再持證明聯至指定據點兌獎
```

若買方使用的是**未歸戶**的會員載具中獎，電子發票實施作業要點第24點規定，營業人應於統一發票開獎日翌日起**10日內**以簡訊、電子郵件或其他適當方式通知中獎人，並提供電子發票證明聯交付中獎人作為兌獎憑證；相關通知紀錄與未獲領取的證明聯，至少要保存1年。電子發票證明聯以列印一次為限，一旦列印後遺失不得補發、也無法再次領獎，因此財政部積極鼓勵買方直接綁定匯款帳戶。

## 營業人的異常賠償責任

若因營業人開立電子發票時發生重複開立、自行作廢、漏傳至整合服務平台等錯誤，或其他可歸責於營業人的事由，導致買受人無法領獎、誤領或溢領獎金，該筆未能領取、誤領或溢領的獎金，應由營業人於查獲日起**15日內**辦理賠償或償還。

## 傳產實例

工廠直營門市的收銀系統若讓消費者用手機條碼索取雲端發票，門市人員應留意：若消費者的載具屬於門市自家發行、尚未歸戶到共通性載具的會員卡，一旦該筆發票中獎，門市有義務在開獎後10日內主動通知並提供中獎證明聯，不能因為「消費者自己沒去查」而免除通知責任。

## 常見錯誤

- 誤以為雲端發票中獎後系統會自動處理一切，忽略營業人對未歸戶載具中獎負有主動通知義務
- 電子發票證明聯已列印一次後，遺失時嘗試要求店家補印，忽略證明聯以列印一次為限、遺失不予補發的規定
- 因系統設定錯誤導致發票重複開立或漏傳而使買受人無法領獎，卻未在查獲日起15日內辦理賠償，延誤處理時限
