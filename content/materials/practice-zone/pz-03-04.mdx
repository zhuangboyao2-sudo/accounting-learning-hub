## 一筆薪資給付牽動的四種負擔

前三節分別介紹了勞保、健保、勞退三種費用的費率與負擔比例，本節把它們整合到實際的薪資分錄中。發放薪資時，公司要同時處理：

```
一、薪資費用（毛支薪資，即員工稅前應領總額）
二、代扣員工負擔部分：勞保自付額、健保自付額、員工自願提繳勞退（若有）、所得稅扣繳額
三、雇主負擔部分：勞保雇主負擔額、職災保險費（全額雇主負擔）、健保雇主負擔額、勞退雇主提繳6%
四、實際入帳／發放金額：毛支薪資減代扣項目後，員工實際領到的淨額
```

其中「雇主負擔部分」不會出現在員工薪資單的減項中，而是公司**另外認列的薪資相關費用**，不影響員工實拿金額，但會增加公司的用人成本。

## 計算範例

某員工月薪資（含經常性給與）30,000元，勞保／健保投保薪資對應級距均為30,300元，勞退月提繳工資級距亦為30,300元，員工未自願提繳勞退，當月所得稅依扣繳稅額表計算為0元（未達起扣標準）：

```
勞保雇主負擔 = 30,300 × 12.5% × 70% ≈ 2,651元
勞保員工自付 = 30,300 × 12.5% × 20% ≈ 758元
健保雇主負擔 = 30,300 × 5.17% × 60% ×（1＋0眷屬）≈ 940元
健保員工自付 = 30,300 × 5.17% × 30% ×（1＋0眷屬）≈ 470元
勞退雇主提繳 = 30,300 × 6% ≈ 1,818元（無勞退員工自付，因未自願提繳）
```

## 發放薪資之分錄

```
借：薪資費用                30,000
貸：應付薪資（或銀行存款）      28,772
貸：代扣勞保費－員工              758
貸：代扣健保費－員工              470
（所得稅扣繳額為0，本例暫不列示）
```

## 認列雇主負擔之薪資相關費用分錄

```
借：勞健保費用（薪資相關費用）    3,591（勞保雇主負擔2,651＋健保雇主負擔940）
借：退休金費用                  1,818
貸：應付勞保費－雇主            2,651
貸：應付健保費－雇主              940
貸：應付退休金                  1,818
```

## 繳納代收款項與雇主負擔款項

```
借：代扣勞保費－員工              758
借：代扣健保費－員工              470
借：應付勞保費－雇主            2,651
借：應付健保費－雇主              940
借：應付退休金                  1,818
貸：銀行存款                   6,637
```

實務上勞保費、健保費是合併向勞保局繳納一張帳單（含員工自付＋雇主負擔），退休金則是繳交至勞保局的個人退休金專戶系統，記帳士應分別設置「應付」科目追蹤代扣與雇主負擔部分，月底核對是否與勞保局、健保署的實際扣款金額一致。

## 常見錯誤

- 誤將雇主負擔的勞健保費、勞退提繳金額與員工自付部分混記在同一個「代扣款」科目下，導致月底無法清楚區分公司實際負擔的用人成本與代扣代繳金額
- 計算薪資費用時只計入毛支薪資，忽略雇主負擔的勞健保費、職災保險費、勞退提繳都是額外的薪資相關費用，會顯著墊高實際用人成本（一般估算約為毛薪資的1.1至1.2倍）
- 誤以為員工自願提繳勞退部分應計入薪資費用增加公司成本，忽略這是從員工薪資中扣除、存入其個人帳戶，不增加公司的用人成本，只是代為轉存
