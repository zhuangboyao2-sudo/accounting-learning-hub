## 薪資所得扣繳：兩種方式擇一

扣繳義務人給付薪資所得，可依規定選擇下列方式之一辦理扣繳：

**方式一：查薪資所得扣繳稅額表**——依受領人有無配偶、受扶養親屬人數及全月薪資數額，查表列應扣稅額；未達表列起扣標準者，免予扣繳。這是多數傳產工廠採用的方式，因為考量了員工的家庭負擔（配偶、扶養親屬人數越多，起扣門檻越高）。

**方式二：按全月給付總額扣取<TaxParam path="withholdingRates.salary.rate" format="percent" />**——不論員工有無配偶或扶養親屬，一律按給付總額固定比率扣繳，計算簡便但不考慮個別員工的家庭負擔。

員工到職時可與雇主約定採用哪種方式；同一員工同一年度內原則上應維持同一種方式，避免扣繳金額忽高忽低造成困擾。

## 執行業務報酬扣繳

工廠常見給付執行業務者（如會計師、記帳士、律師、設計師、工程顧問）之報酬，依規定：

```
一、境內居住者：按給付報酬扣繳10%
二、非境內居住者：按給付報酬扣繳20%
```

給付稿費、版稅、樂譜、作曲、編劇、漫畫、講演鐘點費等執行業務報酬，境內居住者同樣按10%扣繳——但「租稅申報實務」科目提過，這類所得全年合計不超過18萬元享免稅優惠（那是所得人結算申報時的免稅額度，與給付當下扣繳與否是兩個獨立層次：扣繳義務人給付時仍應先依10%扣繳，所得人於次年結算申報時再依免稅規定計算實際應納稅額，若扣繳稅款多繳，可透過結算申報退還）。

## 非即期支票給付之扣繳時點

實務上工廠常以支票支付執行業務者報酬。若以非即期支票給付，扣繳時點為**支票所載發票日**，扣繳義務人應於發票日當天辦理扣繳，而非等到支票兌現日——這是常見的實務陷阱：會計人員容易誤以為要等對方兌現才處理扣繳，實際上發票日一到就已經產生扣繳義務。

## 計算範例

某工廠給付會計師記帳費用每月1萬元，全年（10月至12月，共3個月）記帳費用3萬元，12月30日一次開立支票支付：

```
扣繳稅款 = 30,000 × 10% = 3,000元
```

扣繳義務人應於支票發票日（12月30日）當天扣繳，並最遲於次年1月10日前將代扣稅款繳交國庫。

某員工全月薪資5萬元，已婚且無子女扶養，若查薪資所得扣繳稅額表未達起扣標準，則免予扣繳；若改按給付總額扣取5%，則：

```
扣繳稅款 = 50,000 × 5% = 2,500元
```

兩種方式計算結果可能不同，這也是為什麼實務上多數企業傾向為員工選用「查表」方式，通常對受薪家庭較為有利。

## 常見錯誤

- 誤以為執行業務報酬扣繳與該所得人結算申報時的免稅額度（如稿費18萬元免稅）互相牴觸，忽略給付當下仍應先依規定扣繳，免稅是所得人事後結算申報時才適用的計算結果
- 以非即期支票給付執行業務報酬時，誤以為要等支票兌現才辦理扣繳，忽略扣繳時點是支票所載發票日
- 誤以為薪資扣繳只能用「查表」一種方式，忽略「按全月給付總額扣取固定比率」也是法定方式之一，兩者可依約定選擇
