## 「單筆」如何認定

股利補充保費以「單筆」給付金額判斷是否達2萬元門檻，但投資人常同時持有多家公司股票、且各公司發放時間不一，「單筆」的認定規則如下：

```
一、不同公司發放之股利：即使發放日相同，仍分別視為不同筆，各自判斷是否達2萬元門檻
二、同一公司現金股利與股票股利，發放日不同：金額分開計算，各自判斷門檻
三、同一公司現金股利與股票股利，發放日相同：金額合併計算，合計達2萬元即整筆課徵
```

實務上要判斷是否觸發補充保費，第一步永遠是先確認「這是哪一家公司、哪一個基準日」的股利，而不是把股票帳戶裡當期收到的所有股利金額籠統加總。

## 股票股利之補充保費計算

股票股利（配股）雖然不像現金股利直接拿到現金，但依所得稅法認定股票股利同屬股利所得，健保法亦明定股利所得應計收補充保費，計算公式：

```
股票股利補充保費 = 股票股數 × 面額10元 × 2.11%
```

例如某公司配發股票股利2,000股（面額10元），現金股利0元：

```
股票股利換算金額 = 2,000 × 10 = 20,000元（達門檻）
補充保費 = 20,000 × 2.11% = 422元
```

## 現金股利與股票股利同基準日的合併計算

若同一公司同一基準日同時配發現金股利與股票股利：

```
補充保費 = （股票股數 × 面額10元 ＋ 現金股利金額）× 2.11%
```

例如某公司同一基準日配發現金股利15,000元、股票股利1,000股：

```
合計計費金額 = 15,000 ＋（1,000 × 10）= 25,000元（達門檻）
補充保費 = 25,000 × 2.11% = 527.5元
```

## 現金股利不足支應股票股利補充保費時

若當次僅發放股票股利（無現金股利），或現金股利金額不足以支付整筆補充保費，扣費義務人應通知保險對象（股東），差額由健保署於次年度另行收取——這代表公司作為扣費義務人，若遇到現金不足扣取的情形，不能自行吸收或忽略，而是要依規定通知股東並協助後續處理。

## 常見錯誤

- 誤以為同一天收到的所有股利（不同公司）應合併計算是否達2萬元門檻，忽略不同公司發放之股利一律分別計算
- 計算股票股利之補充保費時，誤用市價而非面額10元計算換算金額
- 誤以為只要沒有現金股利入帳，股票股利就不會被課徵補充保費，忽略股票股利仍須依股數×面額換算後計算，現金不足時由健保署於次年另行收取差額
