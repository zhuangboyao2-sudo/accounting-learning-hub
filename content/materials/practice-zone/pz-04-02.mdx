## 個人一整年須留意的時點

雖然綜所稅結算申報只在每年5月一次性完成，但個人（尤其是有股利、租金、執行業務收入等非薪資所得的人）全年實際上有多個時點需要留意：

```
2月15日至3月16日：可申請「分戶資料限制下載」或「不提供全部扣除額資料」
2月10日前：應收到雇主、金融機構、營利事業等開立的扣繳憑單或免扣繳憑單
4月28日起：財政部電子申報繳稅服務系統開放使用憑證下載當年度所得及扣除額資料
5月1日至5月31日：辦理綜合所得稅結算申報
```

## 申報前的憑證核對清單

個人辦理結算申報前，建議依序核對：

```
一、薪資扣繳憑單：確認全年給付總額與已扣繳稅額是否與自己實際領到的薪資紀錄一致
二、股利憑單：確認股利所得金額，並事先試算合併計稅與分開計稅何者較有利
三、執行業務報酬扣繳憑單：確認是否所有兼職收入來源都已列入
四、利息、租金所得資料：核對是否有漏列或誤植
五、二代健保補充保費扣費證明：這不是所得稅的一部分，但可作為核對股利、租金等所得是否正確申報之輔助依據
```

## 未收到憑單的因應方式

若給付單位遺漏開立扣繳憑單或免扣繳憑單，個人仍有法定的所得申報義務，不能以「沒收到憑單」為由不申報。實務上可透過財政部電子申報系統下載的歸戶所得資料進行核對，若歸戶資料也沒有該筆所得，應主動聯繫給付單位確認是否已依規定申報，必要時可自行依實際所得申報並保留相關給付證明。

## 常見時程誤解

- 誤以為5月才需要開始準備申報資料，忽略部分準備工作（如分戶限制下載申請）須在2-3月間就先辦理
- 誤以為沒收到扣繳憑單就代表不用申報該筆所得，忽略只要有所得就有申報義務，可透過歸戶資料下載核對
- 誤以為股利所得計稅方式一經申報就無法更改，忽略在核課確定前，若發現計稅方式選擇有誤，仍可依規定申請更正

## 常見錯誤

- 錯過2月15日至3月16日的分戶限制下載申請期間，導致當年度無法依個人隱私需求限制資料下載範圍
- 誤以為只要記得5月申報期限即可，忽略提前核對憑單、確認股利計稅方式等準備工作應提前於4月底所得資料開放下載後就開始進行
- 忽略二代健保補充保費扣費證明與所得稅扣繳憑單是兩份不同的文件，核對申報資料時未一併檢視
