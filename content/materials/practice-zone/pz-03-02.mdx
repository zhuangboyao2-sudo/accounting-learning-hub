## 健保費率與負擔比例

現行全民健康保險費率為<TaxParam path="nationalHealthInsurance.rate" format="percent" />。第一類被保險人（有一定雇主之受僱者）之負擔比例：

```
雇主負擔：60%
員工負擔：30%
政府補助：10%
```

## 眷屬健保費：本人＋眷屬人數一併計費

與勞保不同，健保費計算時須將**依附投保的眷屬人數**一併納入：

```
雇主負擔額 = 月投保薪資 × 5.17% × 60% ×（員工本人1人＋眷屬人數）
員工自付額 = 月投保薪資 × 5.17% × 30% ×（本人＋眷屬人數）
```

眷屬計費有一項重要的上限規定：**眷屬人數超過3人者，一律以4人（本人1人＋眷屬3人）計算**，不會因為眷屬人數再多而持續增加保費。這是為了避免多子女或多受扶養親屬的家庭負擔過重的一項保護性上限。

## 計算範例

員工月投保薪資30,300元，依附2名眷屬（本人＋眷屬合計3人）：

```
雇主負擔 = 30,300 × 5.17% × 60% ×（1＋2）≈ 2,822元
員工自付 = 30,300 × 5.17% × 30% ×（1＋2）≈ 1,411元
```

若該員工依附5名眷屬，因超過3人上限，仍以4人（本人＋3人）計算，不是以6人（本人＋5人）計算：

```
雇主負擔 = 30,300 × 5.17% × 60% × 4 ≈ 3,762元
員工自付 = 30,300 × 5.17% × 30% × 4 ≈ 1,881元
```

## 按整月計算，與勞保按天計算不同

健保費不論月初或月中到職，一律按「整個月」計算，不像勞保費按實際加保天數比例計算。實務規則：

```
員工月底離職：由當月（現任）雇主負擔並扣繳當月健保費
員工月中離職：由員工下一個投保單位（新雇主）負責當月健保費，原雇主免繳當月保費
```

這代表工廠計算離職員工當月人事成本時，要先確認離職日是月初、月中還是月底，才能正確判斷該由誰負擔當月健保費，不能一律比照勞保費「按天數比例分攤」處理。

## 投保薪資級距

現行健保投保薪資級距下限<TaxParam path="nationalHealthInsurance.insuredSalaryRange.min" />元、上限<TaxParam path="nationalHealthInsurance.insuredSalaryRange.max" />元，與勞保的級距上下限不同（勞保下限較低、上限也較低），實務作業時兩者級距對照表須分開查閱，不能混用同一張級距表。

## 常見錯誤

- 誤以為健保費和勞保費一樣按實際加保天數比例計算，忽略健保費不論月初或月中到職，一律按整月計算
- 眷屬人數超過3人時，誤以實際眷屬人數（如5人）計算保費，忽略計費人數上限為4人（本人＋眷屬3人）
- 誤用勞保投保薪資級距去計算健保費，忽略兩者的投保薪資級距上下限並不相同
