## 勞保由三個險種組成

實務上俗稱的「勞保費」，其實是三個法源不同的保險合併投保、合併扣款：

```
一、普通事故保險（勞工保險條例）：生育、傷病、失能、老年、死亡給付
二、就業保險（就業保險法）：失業給付、提早就業獎助津貼、職業訓練生活津貼、育嬰留職停薪津貼
三、職業災害保險（勞工職業災害保險及保護法）：職災傷病、醫療、失能、死亡給付
```

前兩者（普通事故＋就業保險）合稱「勞保費率」，現行為<TaxParam path="laborInsurance.generalRate" format="percent" />（其中就業保險費率<TaxParam path="laborInsurance.employmentInsuranceRateIncluded" format="percent" />已包含在內）；職業災害保險費率則依行業別風險程度分別訂定，**由雇主全額負擔**，不算入前述12.5%的分擔比例計算。

## 勞保（含就保）的負擔比例

第一類被保險人（有一定雇主之受僱者，也就是傳產工廠一般員工）之勞保費負擔比例：

```
雇主負擔：70%
員工負擔：20%
政府補助：10%
```

計算公式：

```
雇主負擔額 = 月投保薪資 × 12.5% × 70%
員工自付額 = 月投保薪資 × 12.5% × 20%
```

例如員工月投保薪資30,300元：

```
雇主負擔 = 30,300 × 12.5% × 70% ≈ 2,651元
員工自付 = 30,300 × 12.5% × 20% ≈ 758元
```

## 投保薪資級距

現行勞保投保薪資級距下限<TaxParam path="laborInsurance.insuredSalaryRange.min" />元、上限<TaxParam path="laborInsurance.insuredSalaryRange.max" />元。這代表即使員工實際月薪超過上限，勞保費仍以上限級距計算，不會無上限增加；月薪未達下限者（如部分工時），則依實際工資對應之級距投保。

## 職業災害保險：雇主全額負擔

職業災害保險費率依行業風險程度分級（例如一般辦公室行政人員費率較低，具高風險作業性質的製造業、營建業費率較高），且**全額由雇主負擔**，員工不須分攤。這是傳產製造業成本規劃上容易被低估的一項——工廠類行業的職災費率通常高於服務業，記帳士協助客戶編列人事成本預算時，應個別確認公司所屬行業的職災保險費率，不能直接套用一般辦公室的低費率估算。

## 按日計算 vs 按月計算

勞保費是按「天」計算，員工月中到職或離職，均需按實際加保天數比例計算；這與健保費按「整月」計算的規則不同（詳見下一節）。例如員工12月13日離職，該月僅計收13天勞保費：

```
雇主負擔 = 月投保薪資 × 12.5% × 13/30 × 70%
```

## 常見錯誤

- 誤以為勞保費率12.5%就是全部保費負擔，忽略職業災害保險費率是另外依行業別計算、且由雇主全額負擔，不含在12.5%之內
- 誤以為勞保費和健保費一樣按整月計算，忽略勞保費是按實際加保天數比例計算
- 誤用一般服務業的低職災費率去估算製造業或營建業的人事成本，忽略職災費率依行業風險程度分級，差異可能不小
