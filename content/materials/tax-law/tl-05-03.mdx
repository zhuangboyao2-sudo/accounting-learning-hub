## 一般營業人稅率

第10條規定，營業稅稅率除本法另有規定外，最低不得少於5%，最高不得超過10%，實際徵收率由行政院定之。現行一般營業人適用之徵收率為<TaxParam path="businessTax.generalRate" format="percent" />，這也是傳產製造業、買賣業最常接觸的稅率。

## 金融保險業之特別稅率（第11條）

```
一、經營非專屬本業之銷售額：適用第10條一般稅率
二、銀行業、保險業經營銀行、保險本業銷售額：稅率5%（保險業本業銷售額應扣除財產保險自留賠款；但保險業再保費收入稅率為1%）
三、前二款以外之銷售額（如信託投資業、證券業、期貨業、票券業、典當業之本業銷售額）：稅率2%
```

## 特種飲食業之稅率（第12條）

```
一、夜總會、有娛樂節目之餐飲店：稅率15%
二、酒家及有陪侍服務之茶室、咖啡廳、酒吧等：稅率25%
```

## 小規模營業人之稅率（第13條）

第13條規定，<Term id="small-scale-business">小規模營業人</Term>、視覺功能障礙者經營之按摩業，及其他經財政部規定免予申報銷售額之營業人，稅率為1%；農產品批發市場之承銷人及銷售農產品之小規模營業人，稅率為0.1%。小規模營業人指規模狹小、經稽徵機關依查得資料核定銷售額課徵（而非自行申報銷售額）的營業人，例如街邊小吃攤、傳統雜貨店，與需自行申報401申報書、可扣抵進項稅額的一般營業人（加值型）待遇不同——小規模營業人是非加值型，稅率直接乘銷售額，不得扣抵進項稅額。

## 銷售額之計算（第16條）

第16條規定，銷售額為營業人銷售貨物或勞務所收取之全部代價，包括在價額外收取之一切費用，**但本次銷售之營業稅額不在其內**——也就是銷售額是「未稅價」。若貨物另應課徵貨物稅、菸酒稅或菸品健康福利捐，銷售額還要加計這些稅捐金額在內（先加計貨物稅等，再乘營業稅率）。

例如：小明的工廠銷貨未稅價10萬元，外加5%營業稅5,000元，客戶共付10.5萬元；這裡的銷售額是10萬元（未稅價），5,000元是<Term id="output-tax">銷項稅額</Term>，兩者分開計算、分開記載於統一發票。

## 當期應納或溢付稅額之計算（第15條）

第15條規定，營業人當期<Term id="output-tax">銷項稅額</Term>扣減<Term id="input-tax">進項稅額</Term>後之餘額，為當期應納或溢付營業稅額；銷貨退回或折讓應於發生當期的銷項稅額中扣減，進貨退出或折讓應於發生當期的進項稅額中扣減。

```
當期應納（或溢付）稅額 ＝ 當期銷項稅額 － 當期進項稅額
```

若銷項小於進項（例如外銷比例高、進項大量累積的傳產工廠），差額為溢付稅額，可依規定申報留抵下期或退稅。

## 進項稅額不得扣抵之情形（第19條）

第19條規定，下列進項稅額**不得**扣抵銷項稅額：

```
一、購進之貨物或勞務未依規定取得並保存合法憑證者
二、非供本業及附屬業務使用之貨物或勞務（協助國防建設、慰勞軍隊、對政府捐獻者除外）
三、交際應酬用之貨物或勞務
四、酬勞員工個人之貨物或勞務
五、自用乘人小汽車
```

此外，專營第8條免稅貨物或勞務的營業人（如醫院、學校），其進項稅額不得申請退還，只能認列為成本費用（呼應上一節零稅率與免稅的區別）。

## 常見錯誤

- 計算銷售額時把營業稅額也算進去，忽略第16條規定銷售額不含本次銷售之營業稅額（銷售額是未稅價，不是含稅價）
- 誤以為所有金融業都適用5%稅率，忽略第11條區分本業（5%或1%）與非本業（適用一般稅率）、及其他業務（2%）
- 誤以為交際應酬用的進項稅額只是「不建議」扣抵，忽略第19條規定這是法律**明文禁止**扣抵的項目，即使取得合法統一發票也不能扣抵
- 混淆小規模營業人（非加值型，稅率1%直接乘銷售額，不得扣抵進項）與一般營業人（加值型，銷項減進項）的計稅方式
