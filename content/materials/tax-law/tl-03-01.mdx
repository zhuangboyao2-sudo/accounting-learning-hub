## 計徵公式

所得稅法第13條規定，個人之綜合所得稅，就個人綜合所得總額，減除免稅額及扣除額後之綜合所得淨額計徵之：

```
綜合所得總額 − 免稅額 − 扣除額 = 綜合所得淨額
綜合所得淨額 × 適用稅率 − 累進差額 = 應納稅額
```

## 十類所得

第14條規定，個人綜合所得總額以全年下列十類所得合併計算：

```
第一類：營利所得（股利、盈餘分配等）
第二類：執行業務所得（律師、會計師、記帳士等執業收入減必要費用）
第三類：薪資所得
第四類：利息所得
第五類：租賃所得及權利金所得
第六類：自力耕作、漁、牧、林、礦之所得
第七類：財產交易所得
第八類：競技、競賽及機會中獎之獎金或給與
第九類：退職所得（退休金、資遣費等）
第十類：其他所得
```

## 執行業務所得的帳簿義務

第14條第1項第二類規定，執行業務者至少應設置日記帳一種，詳細記載業務收支項目；業務支出應取得確實憑證；帳簿及憑證最少應保存<Term id="practitioner-records-retention">五年</Term>——這與商業會計法、稅捐稽徵法規定的五年保存期限一致，記帳士本人的執業所得也適用此規定。

## 常見錯誤

- 誤以為綜合所得稅是對「總收入」直接課稅，忽略須先減除免稅額與扣除額，以「淨額」計徵
- 誤以為十類所得中的「其他所得」是隨意認定的概括項目，實際上第10類明確定義為「不屬於上列各類之所得」，且部分特定所得（如告發檢舉獎金、結構型商品交易所得）依規定另採分離課稅不併計綜合所得總額
