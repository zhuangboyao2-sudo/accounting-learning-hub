## 現銷、賒銷與營業稅

台灣的一般營業人銷售貨物或勞務，依規定要開立統一發票，向買受人收取<TaxParam path="businessTax.generalRate" format="percent" />的<Term id="output-tax">銷項稅額</Term>。銷售交易依收款方式不同，分錄略有差異：

**現銷（當場收現金）**

```
借：現金              105,000
    貸：銷貨收入                100,000
    貸：應付營業稅               5,000
```

**賒銷（客戶約定日後付款）**

```
借：應收帳款           105,000
    貸：銷貨收入                100,000
    貸：應付營業稅               5,000
```

兩者唯一的差別，是借方科目從「現金」換成「應收帳款」——依<Term id="accrual-basis">權責發生制</Term>，只要貨物已交付、收款權利已確定，就要認列收入，不需要等到實際收到現金。

## 現購、賒購與進項稅額

反過來，公司向供應商採購原料或商品，支付的營業稅稱為<Term id="input-tax">進項稅額</Term>，這筆稅額日後可以拿來扣抵銷項稅額，減少要繳的營業稅。

**現購（當場付現金）**

```
借：存貨               50,000
    應收退稅（進項稅額）  2,500
    貸：現金                    52,500
```

**賒購（約定日後付款）**

```
借：存貨               50,000
    應收退稅（進項稅額）  2,500
    貸：應付帳款                52,500
```

> 實務上進項稅額通常會併入「應付營業稅」科目一起軋抵，而不是另立「應收退稅」科目；這裡分開列示是為了讓你看清楚進項稅額本質上是「未來可扣抵的權利」，跟存貨成本要分開處理，不能把稅額也算進存貨成本裡。

## 傳產情境：買賣業的一次採購與銷售循環

阿華經營一間五金批發行，本月發生：

1. 向上游廠商賒購一批工具，貨款 20 萬元、進項稅額 1 萬元，約定月底付款
2. 將其中一部分工具現銷給零售商，貨款 8 萬元、銷項稅額 4,000 元，客戶當場刷卡付款

交易 ① 分錄：

```
借：存貨              200,000
    應收退稅（進項稅額）  10,000
    貸：應付帳款                210,000
```

交易 ② 分錄：

```
借：現金               84,000
    貸：銷貨收入                80,000
    貸：應付營業稅               4,000
```

這兩筆交易分別產生了 1 萬元進項稅額與 4,000 元銷項稅額，雙月申報營業稅時，銷項稅額減進項稅額才是實際要繳（或溢付）的稅款——這部分會在租稅申報實務科目與營業稅試算器中詳細說明。

## 常見錯誤

- 把應收帳款、應付帳款的科目寫反：應收帳款是資產（別人欠我們），應付帳款是負債（我們欠別人）
- 把含稅總價全部記進銷貨收入或存貨成本，沒有把營業稅額分開列示
- 現購時把進項稅額也算進存貨成本，導致存貨成本虛增
