## 複習：直線法

上一章介紹過<Term id="straight-line">直線法</Term>：每年折舊費用固定，公式為（成本－殘值）÷耐用年限。本節用同一個例子，比較另外兩種常見的折舊方法。

阿華買入一台新機器，成本 500,000 元，估計可使用 5 年，估計殘值 50,000 元，可折舊總額 = 500,000 − 50,000 = 450,000 元。

## 年數合計法

<Term id="sum-of-years-digits">年數合計法</Term>是一種加速折舊法：把可折舊總額依「剩餘年數」的比例分攤，前期折舊費用較高、後期較低。

```
年數合計 = 5 + 4 + 3 + 2 + 1 = 15（或用公式 5×6÷2 = 15）

第1年折舊 = 450,000 × 5/15 = 150,000 元
第2年折舊 = 450,000 × 4/15 = 120,000 元
第3年折舊 = 450,000 × 3/15 = 90,000 元
第4年折舊 = 450,000 × 2/15 = 60,000 元
第5年折舊 = 450,000 × 1/15 = 30,000 元
```

五年合計 150,000+120,000+90,000+60,000+30,000 = 450,000 元，正好等於可折舊總額。年數合計法適合前期使用效率高、後期效率下降或技術快速淘汰的資產（如電腦設備）。

## 生產數量法

<Term id="units-of-production">生產數量法</Term>按資產的實際使用量（如機器的產量、運輸設備的行駛里程）分攤折舊，而不是以時間為基礎：

```
每單位折舊 = 可折舊總額 ÷ 估計總產量
          = 450,000 元 ÷ 90,000 件 = 5 元／件

若當年度生產 18,000 件：
當年折舊費用 = 18,000 件 × 5 元 = 90,000 元
```

生產數量法最貼近資產實際耗損，適合產量隨景氣波動較大的生產設備。

## 三法比較與適用情境

- **直線法**：每年折舊金額固定，適合平均耗損的資產（如辦公設備、建築物）
- **年數合計法**：前期折舊較高，適合前期使用效率高、技術快速淘汰的資產
- **生產數量法**：依實際產出計算，最貼近實際耗損，適合產量波動大的生產設備

## 常見錯誤

- 把「年數合計」誤算成年限本身，而不是 1+2+…+n 的加總
- 生產數量法忘記先算「每單位折舊」，直接把年度產量乘以資產總成本
- 誤以為可以任意變更折舊方法來調整當年度損益，忽略一致性原則
