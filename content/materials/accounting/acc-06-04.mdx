## 現金短溢的意義

零用金或現金盤點時，若實際現金數量與帳列金額不符，短少或溢餘都應立即查明並入帳，暫時掛在<Term id="cash-over-short">現金短溢</Term>科目。

## 現金短少

阿華月底盤點零用金，帳列應有 10,000 元，實際只剩 9,700 元，短少 300 元：

```
借：現金短溢                300
    貸：零用金                     300
```

## 現金溢餘

若盤點結果是實際金額比帳列金額多，方向則相反：

```
借：零用金                  X
    貸：現金短溢                   X
```

## 原因查明後的處理

若短溢原因後續查明（例如找到遺漏入帳的憑證），應更正分錄，把現金短溢沖回並改列正確科目；若查無原因，現金短溢科目餘額於期末結轉時，列入其他費用（短少）或其他收入（溢餘）。

## 常見錯誤

- 短溢金額原因不明時，就勉強分攤到某個費用科目，而不是先掛「現金短溢」暫記
- 忘記現金短溢科目具有暫記性質，查明原因後應該更正轉列正確科目，而不是放著不管
