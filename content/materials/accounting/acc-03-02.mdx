## 預付費用的攤銷

延續上一節阿華預付一年火險保費 12 萬元的例子：

**12/1 支付當下**

```
借：預付費用          120,000
    貸：現金                  120,000
```

**12/31 期末調整（認列當月份攤銷金額）**

```
借：保險費用           10,000
    貸：預付費用               10,000
```

調整後，「保險費用」認列了這個月真正受益的 1 萬元（120,000 ÷ 12 個月），「預付費用」科目餘額則剩下 110,000 元，代表還有 11 個月份的保障尚未過期，仍然是一項資產。

## 應付費用（應計費用）

上一章已經示範過：電費 8,000 元本月已使用但尚未收到帳單，這正是一筆典型的應計費用調整分錄：

```
借：水電費用            8,000
    貸：應付費用                8,000
```

## 應計收入

阿華把一筆閒置資金 240,000 元借給關係企業，約定年利率 5%，但利息半年才收一次。這個月雖然還沒收到利息，但依權責發生制，這個月賺得的利息仍要先入帳：

```
本月應計利息 = 240,000 × 5% ÷ 12 個月 = 1,000 元

借：應收利息            1,000
    貸：利息收入                1,000
```

## 固定資產折舊

機器設備會隨著使用逐漸耗損，這種耗損要透過**折舊**分攤到各個受益期間，而不是在購買當下一次認列為費用。假設阿華的機器設備 300,000 元，估計可使用 5 年、無殘值，採直線法計算，年折舊費用：

```
年折舊費用 = 300,000 ÷ 5 年 = 60,000 元／年

借：折舊費用           60,000
    貸：累計折舊               60,000
```

<Term id="accumulated-depreciation">累計折舊</Term>是資產的備抵科目，正常餘額在貸方（與一般資產科目相反），用來累積記錄機器設備從購買至今總共折舊了多少。資產負債表上，機器設備會以「原始成本 − 累計折舊 = 帳面價值」的方式列示，讀者可以同時看到原始成本與已耗用的部分。

## 常見錯誤

- 折舊分錄直接貸記「機器設備」科目本身，而不是貸記「累計折舊」，導致資產負債表看不出原始成本
- 應計收入忘記入帳，導致當期收入低估（尤其容易忽略「還沒收到錢」的收入）
- 預付費用一次全部攤銷完畢，而不是按剩餘受益期間逐月攤銷
