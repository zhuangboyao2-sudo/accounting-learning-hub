## 完工轉入製成品

生產完工的產品，從在製品轉入製成品存貨：

```
借：製成品                95,000
    貸：在製品                    95,000
```

## 銷售時結轉銷貨成本

製成品出售時，比照買賣業的永續盤存制，同時結轉銷貨成本。本批餐桌全數以 150,000 元售出：

```
借：應收帳款             150,000
    貸：銷貨收入                 150,000

借：銷貨成本              95,000
    貸：製成品                    95,000
```

## 期末存貨評價

製造業的原料、在製品、製成品三種存貨，依商業會計處理準則第7條，同樣得依其種類或性質採用個別認定法、先進先出法或平均法計算成本，且應以成本與淨變現價值孰低衡量。

## 常見錯誤

- 忘記製成品銷售時要同時結轉銷貨成本，與買賣業永續盤存制的道理相同
- 忽略在製品的淨變現價值估計要考慮完工尚需投入的成本
