## 三張報表如何連動

```
損益表的本期淨利  ──流入──▶  權益變動表的保留盈餘變動
權益變動表的期末保留盈餘餘額  ──流入──▶  資產負債表的權益項下
資產負債表：資產總計 = 負債總計 + 權益總計（會計恆等式）
```

## 串連前面章節的完整範例

**第2節損益表**算出本期淨利 136,000 元。

**第3節權益變動表**用這個淨利計算保留盈餘：

```
期初保留盈餘                    0
＋本期淨利                136,000
－提列法定盈餘公積        (13,600)
－分派股利                (60,000)
＝期末保留盈餘             62,400
```

**第1節資產負債表**的權益項下，保留盈餘正是這個 62,400 元，與普通股股本 1,000,000 元、資本公積 200,000 元加總，得出權益合計 1,262,400 元，最終使資產總計（2,477,400 元）等於負債及權益總計（2,477,400 元）。

## 常見錯誤

- 誤以為三張報表各自獨立、互不影響，忽略損益表淨利會經由權益變動表流入資產負債表
- 資產負債表兩邊金額對不上時，習慣性調整某個科目湊數，而不是回頭檢查交易記錄或報表編製過程哪裡有誤
