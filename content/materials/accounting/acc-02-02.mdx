## 費用類交易

日常營運費用的分錄邏輯很單純：費用科目增加記借方，付款方式（現金或應付）決定貸方。

**現金支付租金**

```
借：租金費用            30,000
    貸：現金                    30,000
```

**尚未支付的水電費（先入帳、月底才繳）**

```
借：水電費用             8,000
    貸：應付費用                 8,000
```

第二筆分錄示範了<Term id="accrual-basis">權責發生制</Term>的實際運用：即使錢還沒付出去，只要費用已經發生（這個月用了電），就要先認列費用與對應的負債（應付費用），下個月實際付款時再沖銷這筆負債。

## 薪資費用

薪資分錄常讓初學者卡關，因為員工實際領到的金額，通常會先被代扣所得稅、勞保費、健保費：

```
借：薪資費用            50,000
    貸：應付薪資（實發）              43,500
    貸：代扣所得稅                    1,500
    貸：代扣勞健保費                  5,000
```

「薪資費用」是公司這個月真正的人事成本 5 萬元；員工實際拿到手的只有淨額，其餘的部分公司會代為扣繳，之後再分別繳給國稅局與勞健保局。代扣稅款與代扣勞健保費的詳細規則，將在租稅申報實務科目說明。

## 資產購置與不得扣抵的進項稅額

購買機器設備等固定資產，原則上支付的進項稅額也能扣抵銷項稅額：

```
借：機器設備            300,000
    應收退稅（進項稅額）  15,000
    貸：應付帳款                315,000
```

但依<Term id="vat-act">加值型及非加值型營業稅法</Term>第 19 條規定，有幾類支出的進項稅額**不得**扣抵銷項稅額，包括：交際應酬用的貨物或勞務、酬勞員工個人的貨物或勞務（如犒賞員工的禮品）、以及自用乘人小汽車（非供銷售或提供勞務使用的九人座以下小客車）。這幾類支出如果取得統一發票，發票上的稅額必須直接併入成本或費用，不能單獨列一筆「進項稅額」科目：

```
' 老闆買了一台自用小客車 100 萬元、稅額 5 萬元，全部不得扣抵
借：運輸設備          1,050,000
    貸：銀行存款              1,050,000
```

## 常見錯誤

- 把員工實發薪資誤記為公司的薪資費用，少計了代扣稅款與勞健保費的部分
- 買了自用小客車或交際應酬支出，仍把發票上的稅額另立「進項稅額」科目扣抵，違反營業稅法第19條
- 應付費用（權責已發生但尚未支付）忘記入帳，導致當期費用低估
