## 為什麼需要試算表

每天記錄大量分錄、過帳到各個帳戶之後，怎麼確定自己沒有記錯？<Term id="trial-balance">試算表</Term>就是為此而生：把某個時點所有會計科目的期末餘額，依「借方欄」「貸方欄」彙總列示成一張表，再加總兩欄——如果借貸雙方合計金額相等，代表至少「借貸金額有沒有寫平」這件事沒有出錯。

## 編製步驟

1. 把每個帳戶（分類帳）算出期末餘額
2. 依科目正常餘額所在方向，把餘額填入試算表的借方欄或貸方欄
3. 分別加總借方欄與貸方欄
4. 檢查兩欄合計是否相等

## 範例：阿華五金批發行的簡易試算表

假設阿華的批發行本期過帳後，各科目餘額如下：

| 科目 | 借方 | 貸方 |
|------|------|------|
| 現金 | 230,000 | |
| 應收帳款 | 120,000 | |
| 存貨 | 180,000 | |
| 機器設備 | 300,000 | |
| 應付帳款 | | 150,000 |
| 應付營業稅 | | 9,000 |
| 股本 | | 600,000 |
| 銷貨收入 | | 280,000 |
| 進貨成本 | 150,000 | |
| 薪資費用 | 45,000 | |
| 租金費用 | 14,000 | |
| **合計** | **1,039,000** | **1,039,000** |

借方合計與貸方合計都是 1,039,000 元，兩邊相等，代表這期的帳務至少在「借貸記錄是否平衡」這件事上沒有問題。下一節會示範：萬一兩欄對不上，該怎麼找出問題所在。

## 試算表能抓到什麼、抓不到什麼

試算表只能檢查「借貸雙方金額是否相等」，這代表：

- **能抓到**：分錄借貸金額不相等、過帳金額抄錯、只過帳一邊忘記過另一邊
- **抓不到**：科目用錯但借貸方向和金額都對（例如把「水電費」誤記成「租金費用」）、整筆分錄漏記（借貸都沒記，自然還是平衡的）、兩個科目的金額剛好都多記或少記相同金額而互相抵銷

所以試算表平衡，只代表記帳「機械上」沒有明顯錯誤，不代表分錄的判斷完全正確——這也是為什麼記帳士的專業判斷仍然重要。

## 常見錯誤

- 把某個科目的餘額漏列在試算表上，導致兩欄都少算了同一筆金額（這種情況試算表仍可能「意外平衡」，需要對照分類帳逐筆核對）
- 把借方餘額的科目誤填進貸方欄（或反之）
- 加總時算錯合計金額
