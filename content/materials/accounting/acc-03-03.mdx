## 從調整後試算表到兩張報表

做完調整分錄、重新編一次<Term id="trial-balance">試算表</Term>（稱為「調整後試算表」），就可以把科目分成兩組，分別編製兩張最基本的財務報表：

- **收入、費用科目** → 編製**損益表**，反映這段期間的經營成果
- **資產、負債、權益科目** → 編製**資產負債表**，反映某個時點的財務狀況

## 損益表：收入減費用

損益表只挑出收入、費用科目，算出這段期間賺了多少錢：

```
銷貨收入                          200,000
減：營業費用
    薪資費用        80,000
    折舊費用        60,000
    保險費用        10,000
    水電費用        15,000
    費用合計                     (165,000)
本期淨利                           35,000
```

## 資產負債表：某個時點的財務狀況

資產負債表挑出資產、負債、權益科目，並把損益表算出的本期淨利，加進<Term id="retained-earnings">保留盈餘</Term>裡（期初保留盈餘 20,000 ＋本期淨利 35,000 ＝ 55,000）：

```
資產                                負債
現金          100,000               應付帳款        40,000
應收帳款       50,000               應付營業稅       5,000
預付費用       10,000               負債合計        45,000
機器設備      300,000
減：累計折舊  (60,000)              權益
                                     股本          300,000
                                     保留盈餘       55,000
                                     權益合計      355,000

資產合計      400,000               負債＋權益合計   400,000
```

資產合計 400,000 元，剛好等於負債＋權益合計 400,000 元——這正是<Term id="accounting-equation">會計恆等式</Term>在報表層次的體現。也可以看出：**損益表算出的本期淨利，是連結損益表與資產負債表的關鍵橋樑**——淨利會讓權益（保留盈餘）增加，這也是為什麼公司賺錢會讓老闆的權益變大。

## 常見錯誤

- 編資產負債表時忘記把本期淨利併入保留盈餘，導致權益金額少列、恆等式不平衡
- 把累計折舊誤列在負債，而不是資產的減項——累計折舊雖然餘額在貸方，但性質上仍是資產科目的備抵項目
- 損益表漏列某個費用科目，導致淨利虛增，連帶讓資產負債表的保留盈餘也跟著虛增
