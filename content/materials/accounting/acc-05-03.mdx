## 處分時的帳務處理三步驟

1. 補提處分當期截至處分日為止的折舊
2. 結算處分日的帳面價值 = 原始成本 − 累計折舊
3. 比較處分價款與帳面價值，認列<Term id="disposal-gain-loss">處分損益</Term>：價款高於帳面價值認列處分利益，價款低於帳面價值認列處分損失

## 處分利益的例子

阿華的機器設備原始成本 300,000 元，累計折舊已提列 240,000 元（帳面價值 60,000 元）。以 70,000 元出售：

```
借：銀行存款              70,000
    累計折舊              240,000
    貸：機器設備                  300,000
        處分固定資產利益            10,000
```

## 處分損失的例子

同一台機器改以 50,000 元出售（售價低於帳面價值 60,000 元）：

```
借：銀行存款              50,000
    累計折舊              240,000
    處分固定資產損失         10,000
    貸：機器設備                  300,000
```

## 報廢（無法出售，直接淘汰）

若資產報廢且無殘值可回收，帳面價值全數認列為處分損失：

```
借：累計折舊              240,000
    處分固定資產損失         60,000
    貸：機器設備                  300,000
```

## 常見錯誤

- 處分前忘記先補提當期折舊，導致帳面價值計算錯誤
- 把處分價款直接當作處分利益，忘記要先扣掉帳面價值才是真正的損益
- 報廢時忘記把剩餘帳面價值全數轉列處分損失
