## 獨資企業的權益科目

獨資企業沒有法定資本額限制，業主的全部利潤都直接歸屬業主，會計上常用兩個科目：<Term id="owner-capital">業主資本</Term>與業主往來（提取）。

## 業主投入資金

阿華獨資經營批發行，期初投入現金 500,000 元：

```
借：現金                 500,000
    貸：業主資本                  500,000
```

## 業主提取

業主提取是指業主從公司拿錢供自己使用（不是公司的薪資費用），會沖減業主權益：

```
借：業主往來              50,000
    貸：現金                      50,000
```

## 期末結帳

期末結帳時，本期損益與業主往來（提取）餘額都要結轉沖銷業主資本。假設本期淨利 200,000 元，加上前面提取的 50,000 元：

```
借：本期損益              200,000
    貸：業主資本                  200,000

借：業主資本               50,000
    貸：業主往來                   50,000
```

## 常見錯誤

- 把業主提取列為公司費用——提取是業主權益的減項，不是公司的費用，獨資企業沒有「支付薪資給業主本人」的概念
- 忘記期末要把業主往來科目結轉沖銷，遺留在權益底下會混淆下期的餘額
