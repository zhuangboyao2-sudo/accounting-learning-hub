## 確定收不回時的沖銷分錄

```
借：備抵壞帳                 X
    貸：應收帳款                    X
```

沖銷時不影響當期損益，因為壞帳費用已經在估列時認列過了，沖銷只是把備抵壞帳與應收帳款同時減少。

阿華確定客戶倒閉，積欠的 8,000 元應收帳款確定收不回：

```
借：備抵壞帳               8,000
    貸：應收帳款                   8,000
```

## 已沖銷帳款事後收回

若原本已沖銷的應收帳款，事後客戶又還錢了，要先「迴轉」沖銷分錄，再認列收現：

```
借：應收帳款               8,000
    貸：備抵壞帳                   8,000

借：現金                   8,000
    貸：應收帳款                   8,000
```

先迴轉沖銷分錄的目的，是讓應收帳款與備抵壞帳的科目餘額維持完整，之後若同一客戶再度發生類似情形，帳務紀錄才會正確。

## 常見錯誤

- 沖銷分錄誤借記壞帳費用，而不是備抵壞帳——壞帳費用在估列時已經認列過，沖銷時不應該重複認列費用
- 已沖銷帳款收回時，直接借記現金、貸記雜項收入，而不是先迴轉沖銷分錄，導致應收帳款與備抵壞帳金額失真
