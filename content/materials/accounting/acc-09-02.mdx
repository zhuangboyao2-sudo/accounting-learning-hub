## 合夥企業的權益結構

合夥企業由兩個以上合夥人共同出資經營，每位合夥人各自有獨立的<Term id="partner-capital">資本帳戶</Term>與往來（提取）帳戶。

## 損益分配

合夥的損益分配依合夥契約約定的比例進行，可能是均分、依出資比例，或依約定的其他方式（例如先給付合夥人薪資後，再分配剩餘損益）。

阿華與阿明合夥經營商行，約定損益均分。年度淨利 100,000 元：

```
借：本期損益              100,000
    貸：阿華資本                   50,000
        阿明資本                   50,000
```

若改依出資比例分配（阿華出資 60%、阿明出資 40%）：

```
借：本期損益              100,000
    貸：阿華資本                   60,000
        阿明資本                   40,000
```

## 常見錯誤

- 沒有依合夥契約約定的分配方式，一律假設均分
- 把各合夥人的資本帳戶混在一起，沒有分別設立獨立帳戶
