## 什麼是應收票據

<Term id="notes-receivable">應收票據</Term>是持有他人簽發的本票或匯票，約定在未來特定日期兌現的債權憑證，比一般應收帳款多了法律上更強的求償力。依商業會計處理準則第15條，資產負債表日同樣應評估應收票據無法收回之金額，提列適當之備抵呆帳。

## 到期價值的計算

```
到期價值 = 票面金額 + 利息（若為附息票據）
利息 = 票面金額 × 年利率 × 期間（以年為單位）
```

阿華收到客戶簽發的一張本票，面額 100,000 元，年利率 6%，期間 3 個月：

```
利息 = 100,000 × 6% × 3/12 = 1,500 元
到期價值 = 100,000 + 1,500 = 101,500 元
```

## 應收票據貼現簡介

企業若在到期日前需要資金週轉，可以把應收票據拿去銀行「貼現」，提前取得現金，銀行則收取貼現利息：

```
貼現息 = 到期價值 × 貼現率 × 貼現期間（貼現日到到期日之天數）
貼現實得金額 = 到期價值 − 貼現息
```

上例票據在到期前 1 個月拿去銀行貼現，貼現率 9%：

```
貼現息 = 101,500 × 9% × 1/12 ≈ 761 元
貼現實得金額 = 101,500 − 761 = 100,739 元
```

## 常見錯誤

- 忘記應收票據的利息計算要以「年」為基礎，換算成實際持有的月數或天數
- 貼現息計算基礎搞錯，誤用票面金額而非到期價值——貼現息應以到期價值為計算基礎，因為銀行承擔的是到期時應收金額的風險
