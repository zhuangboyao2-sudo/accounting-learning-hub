## 什麼是不動產、廠房及設備

依商業會計處理準則第18條，<Term id="ppe">不動產、廠房及設備</Term>指用於商品、農業產品或勞務之生產或提供、出租予他人或供管理目的而持有，且預期使用期間超過一年之有形資產，包括土地、建築物、機器設備、運輸設備、辦公設備及生產性植物等。

## 取得成本的範圍

原始成本包括購買價格、使資產達到預期運作方式之必要狀態及地點之任何直接可歸屬成本（如運費、安裝費、測試費），以及未來拆卸、移除該資產或復原的估計成本；後續成本則包括後續為增添、部分重置或維修該項目所發生之成本。

阿華的批發行購入一台包裝機器，發票價 300,000 元，另外支付運費 5,000 元、安裝調試費 10,000 元：

```
機器設備原始成本 = 300,000 + 5,000 + 10,000 = 315,000 元

借：機器設備             315,000
    貸：銀行存款                315,000
```

## 資本支出與收益支出的劃分

<Term id="capital-expenditure">資本支出</Term>：能增加資產未來經濟效益、延長使用年限或提升產能的支出，應借記資產科目，透過折舊分年攤銷。

<Term id="revenue-expenditure">收益支出</Term>：維持資產正常運作的例行維修、保養費用，應借記當期費用。

阿華的機器設備例行更換潤滑油，花費 2,000 元，屬於收益支出：

```
借：修繕維護費用           2,000
    貸：銀行存款                  2,000
```

同一台機器更換核心零件、延長使用年限 5 年，花費 50,000 元，屬於資本支出：

```
借：機器設備              50,000
    貸：銀行存款                  50,000
```

## 劃分錯誤的影響

- 把資本支出誤記為收益支出（當期費用）：當期費用高估、當期淨利低估、資產金額低估
- 把收益支出誤記為資本支出（資產）：當期費用低估、當期淨利高估、資產金額高估

## 常見錯誤

- 忘記把運費、安裝費等直接可歸屬成本計入資產原始成本，誤列為當期費用
- 把例行保養維修（收益支出）誤資本化，虛增資產、低估當期費用
- 把能延長使用年限或提升產能的重大支出（資本支出）誤認列為當期費用
