## 為什麼要結帳

收入、費用科目只反映「這段期間」發生的事，每個新的會計期間都應該從零開始重新累計，不然本期的銷貨收入會一直跟上期的加在一起、越滾越大，失去意義。這類科目稱為**暫時性科目**，需要在期末透過**結帳分錄**歸零，並把淨額結轉到<Term id="retained-earnings">保留盈餘</Term>。

相對地，資產、負債、權益科目反映的是「某個時點」的狀態，會一直累積延續到下一期，稱為**永久性科目**，不需要結帳歸零。

## 結帳三步驟

延續上一節阿華批發行的例子（銷貨收入 200,000、費用合計 165,000、本期淨利 35,000），結帳分錄依序進行：

**① 結清收入科目**

```
借：銷貨收入          200,000
    貸：本期損益              200,000
```

**② 結清費用科目**

```
借：本期損益          165,000
    貸：薪資費用                80,000
    貸：折舊費用                60,000
    貸：保險費用                10,000
    貸：水電費用                15,000
```

**③ 本期損益結轉保留盈餘**

「本期損益」是結帳過程中暫時使用的清算科目，經過①②兩步後，餘額為 200,000－165,000＝35,000（貸方餘額，代表淨利），這筆餘額要整筆轉入保留盈餘：

```
借：本期損益           35,000
    貸：保留盈餘                35,000
```

三個步驟做完後，銷貨收入與所有費用科目餘額都歸零，本期損益科目也歸零，35,000 元的淨利正式併入保留盈餘（20,000＋35,000＝55,000）。

## 結後試算表

結帳分錄過帳完成後，重新編一次試算表，此時應該只剩下永久性科目（資產、負債、權益），且借貸仍應相等：

| 科目 | 借方 | 貸方 |
|------|------|------|
| 現金 | 100,000 | |
| 應收帳款 | 50,000 | |
| 預付費用 | 10,000 | |
| 機器設備 | 300,000 | |
| 累計折舊 | | 60,000 |
| 應付帳款 | | 40,000 |
| 應付營業稅 | | 5,000 |
| 股本 | | 300,000 |
| 保留盈餘 | | 55,000 |
| **合計** | **460,000** | **460,000** |

這張結後試算表，就是下一個會計期間的起點——收入、費用科目重新從零開始累計，資產、負債、權益科目則延續前一期的結果。

## 常見錯誤

- 忘記把「本期損益」科目結轉到保留盈餘，導致這個暫時科目的餘額懸在半空中，沒有真正歸零
- 結帳分錄的借貸方向寫反：收入科目結清時是借方（收入正常餘額在貸方，結清要記借方），費用科目結清時是貸方
- 結後試算表裡誤留了收入或費用科目，代表結帳分錄沒有做完整
