## 存貨是什麼

<Term id="inventory">存貨</Term>是企業持有、準備在正常營業過程中出售的商品。買賣業（批發、零售）的存貨主要就是已購入尚未售出的商品；製造業則還會有原料、在製品、製成品等更細的分類。本章先以買賣業最常見的情況為主。存貨在資產負債表上列為流動資產。

## 兩種進貨與存貨的記錄制度

<Term id="perpetual-inventory">永續盤存制</Term>：進貨時立即借記存貨科目，每次銷貨的同時也結轉<Term id="cogs">銷貨成本</Term>，帳上隨時可以得知存貨餘額。

<Term id="periodic-inventory">定期盤存制</Term>：進貨時記入「進貨」科目，平時不更動存貨餘額，要到期末透過實地盤點，再用公式反算銷貨成本：

```
期初存貨 + 本期進貨 − 期末存貨 = 銷貨成本
```

現代企業多半採用電腦化的永續盤存制，但定期盤存制的公式仍是理解下一節「存貨成本流動假設」的基礎。

## 永續盤存制下的進貨分錄

阿華的批發行以應付帳款方式進貨 5,000 元：

```
借：存貨               5,000
    貸：應付帳款              5,000
```

## 進貨運費與進貨退回折讓

進貨運費若由買方負擔，屬於使商品達到可出售狀態的必要成本，應計入存貨成本，而不是列為當期費用：

```
借：存貨                 300
    貸：應付帳款（或現金）        300
```

進貨後發現部分商品瑕疵，供應商同意退貨並沖減應付帳款 800 元：

```
借：應付帳款              800
    貸：存貨                   800
```

## 常見錯誤

- 把進貨運費當作當期銷售費用，而不是計入存貨成本
- 誤以為定期盤存制下也能隨時從帳上看到存貨餘額，其實要等實地盤點才能得知
- 誤以為永續盤存制平時不需要盤點——實際上仍需定期盤點，只是目的是核對帳載與實際數量、抓出短缺，而不是為了計算銷貨成本
