## 完整案例：阿華的家具工廠本月成本核算

**原料**：期初原料 20,000 元，本期採購原料 60,000 元，期末原料庫存 15,000 元：

```
本期領用原料（直接原料）= 20,000 + 60,000 − 15,000 = 65,000 元
```

**其他成本要素**：本期直接人工 40,000 元，本期製造費用 25,000 元。

**在製品**：期初在製品 5,000 元，期末在製品 8,000 元：

```
本期生產總成本 = 65,000 + 40,000 + 25,000 + 5,000 = 135,000 元
本期製成品成本 = 135,000 − 8,000 = 127,000 元
```

**製成品**：期初製成品 0 元（新工廠），本期製成品 127,000 元，本期銷售出的製成品成本 100,000 元（部分尚未售出）：

```
期末製成品 = 0 + 127,000 − 100,000 = 27,000 元
```

## 完整分錄整理

```
1. 領用原料：
   借：在製品                65,000
       貸：原料                      65,000

2. 認列直接人工：
   借：在製品                40,000
       貸：應付薪資                  40,000

3. 分攤製造費用：
   借：在製品                25,000
       貸：各項製造費用              25,000

4. 完工轉入製成品：
   借：製成品               127,000
       貸：在製品                   127,000

5. 銷售結轉銷貨成本：
   借：銷貨成本             100,000
       貸：製成品                   100,000
```

## 常見錯誤

- 本期領用原料的計算公式（期初＋採購－期末）算錯，忘記期末未用完的原料要先扣除
- 各步驟之間的存貨科目（原料→在製品→製成品）轉換順序搞混，跳過中間步驟直接結轉
