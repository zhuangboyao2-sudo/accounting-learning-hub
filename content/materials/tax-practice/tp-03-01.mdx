## 營業稅申報書代號總覽

國稅局將營業稅申報書依適用對象分為多種代號，前面「稅務相關法規」科目介紹的401申報流程，實際上只是其中一種：

| 代號 | 適用對象 |
|------|----------|
| 401 | 一般稅額計算，專營應稅（含零稅率）之營業人 |
| 402 | 境外電商營業人 |
| 403 | 一般稅額計算，兼營免稅或特種稅額計算之營業人 |
| 404 | 專營特種稅額計算之營業人（金融保險業、特種飲食業等） |
| 405 | 查定課徵之小規模營業人（免用統一發票、月銷售額低於財政部規定標準），按季由稽徵機關開單繳納 |
| 406 | 租稅規避案件之繳款書 |
| 407 | 非以營利為目的之事業、機關、團體、組織有銷售貨物或勞務者 |
| 408 | 買受人依規定應繳納購買國外勞務營業稅之申報 |

傳產工廠、一般買賣業、服務業最常使用的是401；同時經營應稅與免稅業務（如診所兼賣自費商品、賣場兼營免稅農產品）則要改用403；銀行、保險等專營特種稅額計算業別使用404。**選錯申報書代號**是實務上常見的錯誤來源之一。

## 401申報書：銷項欄位結構

401申報書（一般稅額計算－專營應稅營業人使用）銷項欄依發票種類分列：

```
一、三聯式發票、電子計算機發票：應稅銷售額、稅額
二、收銀機發票（三聯式）及電子發票：應稅銷售額、稅額
三、二聯式發票、收銀機發票（二聯式）：應稅銷售額、稅額
四、免用發票：依規定免用或免開統一發票之銷售額
五、退回及折讓：銷售額及稅額之減項，依「退回（出）或折讓證明單」填列
六、零稅率銷售額：分「非經海關出口應注明文件者」與「經海關出口免附證明文件者」
```

「合計」欄＝前四項加總減退回及折讓；「銷售額總計」欄＝應稅合計＋零稅率合計，並另填明其中固定資產銷售額。

## 401申報書：進項欄位結構

```
一、統一發票扣抵聯：進貨及費用、固定資產（分別列示，減除第19條不得扣抵項目後之餘額）
二、三聯式收銀機發票扣抵聯及一般稅額計算之電子發票
三、載有稅額之其他憑證（含二聯式收銀機發票）
四、海關代徵營業稅繳納證扣抵聯：進口貨物由海關代徵之營業稅憑證（見「稅務相關法規」科目進口貨物營業稅一節）
五、退出、折讓及海關退還溢繳稅款：進項金額及稅額之減項
```

「進項總金額」欄則涵蓋含稅額得扣抵、不得扣抵之憑證及普通收據等憑證所載金額總數——這與「得扣抵進項稅額」是不同欄位，前者是全部進項的總額（供稽核比對用），後者才是實際拿來扣抵銷項稅額的金額。

## 稅額計算區塊：從銷項到應納/留抵稅額

```
① 本期銷項稅額合計
② 得扣抵進項稅額（統一發票扣抵聯＋海關代徵營業稅繳納證扣抵聯等合計）
③ 上期累積留抵稅額
④ 小計 = ②＋③
⑤ 若①＞④：本期應實繳稅額 = ①－④
⑥ 若④＞①：本期申報留抵稅額 = ④－①
```

這正是第15條「當期應納或溢付稅額＝銷項稅額－進項稅額」在申報書上的具體呈現，只是多了「上期累積留抵稅額」一併結轉計算。

## 常見錯誤

- 誤用申報書代號：同時有免稅收入或特種稅額計算收入，卻仍用401申報，未改用403或404
- 混淆「進項總金額」與「得扣抵進項稅額」：前者是全部憑證金額（含不得扣抵部分），後者才是實際可扣抵的金額，兩者不應混為一談
- 忘記零稅率銷售額須依「經海關出口」與「非經海關出口」分欄填列，並準備對應的證明文件
