## 稅額計算的完整流程

「稅務相關法規」科目已個別介紹過免稅額、標準扣除額、列舉扣除額等概念，本節把這些拼圖組合成一份申報書實際跑一次的完整計算流程：

```
① 各類所得合計（薪資、利息、租賃、股利、財產交易所得等，各類所得先減除該類之必要費用或特別扣除額）
② － 免稅額（本人、配偶、受扶養親屬）
③ － 一般扣除額（標準扣除額或列舉扣除額，擇一）
④ － 特別扣除額（薪資所得特別扣除額已於①扣除；另有身心障礙、教育學費、幼兒學前、長照等特別扣除額）
⑤ ＝ 綜合所得淨額
⑥ × 適用稅率級距 － 累進差額 ＝ 應納稅額
⑦ － 扣繳稅額、可扣抵稅額（如股利8.5%抵減稅額）＝ 應退（補）稅額
```

## 完整計算範例

王先生，單身，無受扶養親屬，115年度（2026年）全年薪資收入80萬元，無其他所得，雇主全年已預扣薪資所得稅款12,000元。

**第一步：計算薪資所得**

薪資所得＝薪資收入80萬元－薪資所得特別扣除額<TaxParam path="individualIncomeTax.salaryIncomeSpecialDeduction" />＝800,000－227,000＝**573,000元**。

**第二步：計算綜合所得淨額**

綜合所得總額為573,000元（僅薪資所得，無其他所得類別），減除免稅額（本人）<TaxParam path="individualIncomeTax.personalExemption" />、標準扣除額（單身）<TaxParam path="individualIncomeTax.standardDeductionSingle" />後：

綜合所得淨額＝573,000－101,000－136,000＝**336,000元**。

**第三步：查稅率級距，計算應納稅額**

336,000元落在第一級距（61萬元以下，稅率5%，累進差額0）：

```
應納稅額 = 336,000 × 5% － 0 ＝ 16,800元
```

**第四步：計算應退（補）稅額**

```
應退（補）稅額 = 應納稅額 － 已扣繳稅額 ＝ 16,800 － 12,000 ＝ 應補稅額4,800元
```

王先生今年須補繳4,800元。

## 累進差額的作用

累進稅率若逐級距分段計算會很繁瑣，累進差額就是把「分段計算」簡化成「淨額直接乘最高適用稅率、再減一個固定差額」的捷徑數字。以115年度稅率結構為例：

```
綜合所得淨額           稅率    累進差額
61萬元以下              5%      0
61萬元～138萬元         12%     42,700
138萬元～277萬元        20%     153,100
277萬元～519萬元        30%     430,100
519萬元以上             40%     949,100
```

例如綜合所得淨額100萬元，落在第二級距：應納稅額＝100萬×12%－42,700＝77,300元。若不使用累進差額，須拆成「61萬×5%＋（100萬－61萬）×12%」＝30,500＋46,800＝77,300元，結果相同但計算較繁瑣——累進差額正是把前面級距的稅額差異預先折算好的固定數字。

## 常見錯誤

- 計算綜合所得淨額時忘記先減薪資所得特別扣除額，直接把全年薪資收入當作薪資所得計入總額，導致淨額虛增
- 誤以為累進稅率是「全部所得都按最高級距稅率課稅」，忽略累進差額正是用來還原分級累進的正確稅額
- 忘記已扣繳稅額可以抵減應納稅額，誤以為應納稅額全部都要另外再繳一次
