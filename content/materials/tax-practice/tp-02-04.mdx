## 結算申報期間與流程

會計年度採曆年制之營利事業，應於每年**5月1日至5月31日**止，填具結算申報書並自行計算應納之結算稅額，於申報前自行繳納。整體流程：

```
① 決定當年度採用之申報方式（擴大書審／查帳申報／會計師簽證／所得額標準）
② 依所屬方式計算全年所得額與應納稅額
③ 減除暫繳稅額（9月已繳）、扣繳稅額、投資抵減稅額等
④ 計算應退（補）稅額，先繳納後檢同繳納收據一併申報
⑤ 提交結算申報書及相關附件（財務報表、暫繳稅款繳納收據、扣繳憑單等）
```

## 完整試算範例：擴大書審 vs 查帳申報的選擇

C公司從事製造業，行業代號對應之擴大書審純益率為8%。C公司115年度（2026年）全年營業收入淨額3,000萬元，帳載實際成本費用為2,760萬元（帳載純益率約8%）。9月已依上年度應納稅額二分之一繳納暫繳稅額10萬元。

**若採擴大書審申報：**

```
全年所得額 = 3,000萬 × 8% = 240萬元
應納營利事業所得稅 = 240萬 × 20% = 48萬元
應補稅額 = 48萬 － 暫繳稅額10萬 = 38萬元
```

**若採查帳申報：**

```
課稅所得額 = 3,000萬 － 2,760萬 = 240萬元
應納營利事業所得稅 = 240萬 × 20% = 48萬元
應補稅額 = 48萬 － 暫繳稅額10萬 = 38萬元
```

本例中兩種方式結果相同，因為C公司帳載純益率剛好等於擴大書審純益率標準。但若C公司當年度成本上升、帳載純益率降到5%（實際所得150萬元），則：

```
查帳申報：應納稅額 = 3,000萬×5%×20% = 30萬元（較低，反映實際獲利下降）
擴大書審：仍須以8%純益率計算，應納稅額維持48萬元（較高，未反映實際獲利下降）
```

這說明擴大書審的核心風險：純益率是固定標準，不會因為當年度實際獲利下降而降低稅負；查帳申報才能真實反映當年度盈虧。企業應在申報前，先分別試算兩種方式的稅負，選擇較有利者（若帳務健全、允許採查帳申報）。

## 擴大書審常見的4大錯誤申報樣態

國稅局公告指出，採擴大書審申報的營利事業，最常見的違章樣態包括：

```
一、漏報（或分散）收入：將部分營業收入轉移至他人名下申報，規避完整收入認列
二、虛開發票：開立不實統一發票美化或掩飾實際交易
三、未據實辦理扣繳：給付薪資、租金、佣金等未依規定扣繳稅款
四、未正確填寫公司行業代號：選錯行業代號可能導致誤用不適用之擴大書審純益率
```

這4類錯誤即使表面上採用「較簡便」的擴大書審申報，一旦被查獲，仍會被追繳稅款並處以罰鍰，且擴大書審原本較低的抽查機率也不再適用。

## 結算申報書之退補稅計算

延續本章前述架構，結算申報最終的應退（補）稅額公式為：

```
應退（補）稅額 = 應納稅額 － 暫繳稅額 － 扣繳稅額 － 投資抵減稅額等 － 其他已繳納稅款
```

若結果為正數，須於申報時一併繳納；若為負數，可申請退稅。這與「租稅申報實務」科目第1章介紹的綜所稅退補稅計算邏輯一致，差別只在於營所稅多了「暫繳稅額」這一項扣減。

## 常見錯誤

- 選擇擴大書審申報前未先試算查帳申報的稅負，錯失實際獲利較低年度可能節稅的機會
- 誤以為擴大書審申報抽查機率低就可以隨意處理收入或憑證，忽略漏報收入、虛開發票等4大常見違章樣態仍會被查獲並補稅處罰
- 忘記計算應退（補）稅額時要先扣減9月已繳的暫繳稅額，重複繳納或誤算補稅金額
