## 哪些溢付稅額可以申請退還

第39條規定，營業人申報之溢付稅額，屬於下列情形者，稽徵機關查明後應予退還：

```
一、因銷售第7條規定適用零稅率貨物或勞務而溢付之營業稅
二、因取得固定資產而溢付之營業稅
三、因合併、轉讓、解散或廢止申請註銷登記者，其溢付之營業稅
```

前項以外之溢付稅額，原則上應由營業人留抵應納營業稅（結轉下期繼續扣抵），但情形特殊者，得報經財政部核准退還。這代表：一般營業人單純「進項大於銷項」的溢付稅額，通常不能馬上退現金，而是留著扣抵未來期別的應納稅額；只有外銷型（零稅率）、購置固定資產、或公司要合併轉讓解散的情形，才能真正申請退現金。

## 得退稅限額之計算

401申報書「稅額計算」欄規定，得退稅限額合計之公式為：

```
得退稅限額合計 = 零稅率銷售額合計 × 5% ＋ 固定資產得扣抵之進項稅額
```

本期應退稅額之判斷：

```
若「本期申報留抵稅額」＞「得退稅限額合計」：應退稅額 = 得退稅限額合計（差額轉入本期累積留抵稅額，結轉下期）
若「得退稅限額合計」＞「本期申報留抵稅額」：應退稅額 = 本期申報留抵稅額（本期累積留抵稅額為0）
```

## 計算範例

某外銷型傳產工廠當期銷項稅額80萬元（含零稅率銷售額1,000萬元，稅率0%），進項稅額120萬元（其中含購置一批機器設備之進項稅額15萬元）。

**第一步：計算本期申報留抵稅額**

```
本期申報留抵稅額 = 進項稅額120萬 － 銷項稅額80萬 = 40萬元
```

**第二步：計算得退稅限額合計**

```
得退稅限額合計 = 零稅率銷售額1,000萬 × 5% ＋ 固定資產進項稅額15萬 = 50萬 ＋ 15萬 = 65萬元
```

**第三步：比較並決定應退稅額**

本期申報留抵稅額40萬元 ＜ 得退稅限額合計65萬元，故：

```
本期應退稅額 = 40萬元（全額退還）
本期累積留抵稅額 = 0元
```

若這期進項稅額改為200萬元，留抵稅額變成120萬元（200萬－80萬），大於得退稅限額65萬元，則：

```
本期應退稅額 = 65萬元（受限額限制）
本期累積留抵稅額 = 120萬 － 65萬 = 55萬元（結轉下期）
```

## 應退稅額之領取方式

401申報書提供兩種領取方式：利用存款帳戶劃撥，或領取退稅支票。實務上建議營業人事先向稽徵機關申請「利用存款帳戶劃撥」，可加快退稅到帳速度，避免退稅支票遺失或兌現延誤的問題。

## 常見錯誤

- 誤以為只要進項大於銷項就一定能全額退還現金，忽略一般情形的溢付稅額原則上只能留抵，只有零稅率、固定資產、合併轉讓解散注銷三種情形才能申請退還
- 計算得退稅限額時，忘記零稅率銷售額要乘以5%（一般稅率），而非直接用零稅率銷售額全額當作限額
- 忽略本期應退稅額受限額限制時，超過限額的部分不是消失，而是轉入本期累積留抵稅額，結轉下期繼續適用
