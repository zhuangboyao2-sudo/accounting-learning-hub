## 403申報書的適用對象

403申報書（一般稅額計算－兼營免稅、特種稅額計算營業人使用）適用於：依營業稅法第四章第一節一般稅額計算規定計算稅額，**同時兼營免稅或第四章第二節規定計算稅額**（特種稅額計算，如特種飲食業、金融保險業）之營業人。若是**專營**第四章第二節規定計算稅額之營業人（如專營銀行本業），則不適用403，應改用404申報書。

常見適用情境：診所或藥局兼賣一般商品、賣場兼營免稅農產品銷售、貿易公司同時有應稅商品買賣與土地銷售等——這正是「稅務相關法規」科目提過的「兼營營業人」（見加值型及非加值型營業稅法進階規定章節）在申報書上的實際呈現。

## 銷項欄：比401多出的「免稅銷售額」與「特種稅額」

403申報書銷項欄分為「一般稅額」與「特種稅額」兩大區塊：

**一般稅額欄**：與401大致相同（三聯式發票、收銀機發票、二聯式發票、免用發票、退回及折讓、零稅率銷售額），但**每一類發票欄位都多了「免稅銷售額」一欄**，用來填列該類憑證中屬於免稅之銷售額。

**特種稅額欄**：

```
一、特種飲食業：夜總會、有娛樂節目之餐飲店（15%）；酒家及有陪侍服務之茶室、咖啡廳、酒吧等（25%）
二、銀行業、保險業經營銀行、保險本業收入（5%）
三、銀行業、保險業、信託投資業經營非銀行保險本業之專屬本業收入（2%）
四、再保收入（1%）
五、免稅收入（特種稅額計算部分之免稅銷售額）
```

這些稅率與「稅務相關法規」科目介紹過的第11、12條完全對應——403申報書就是把法規上的稅率規定，落實到實際申報欄位。

## 進項欄：兼營營業人的比例扣抵欄位

403申報書進項欄與401最大的不同，是把「得扣抵進項稅額」改為**「應比例計算得扣抵進項稅額」**——因為兼營營業人的進項稅額，不能像401專營應稅營業人一樣全額扣抵，必須依「兼營營業人營業稅額計算辦法」按比例計算可扣抵部分（呼應「稅務相關法規」科目兼營營業人進項稅額比例扣抵一節）。

```
不得扣抵之進項稅額比例 ≈ 免稅銷售額 ÷ 全部銷售額
```

實務上，403申報書全年最後一期須做「年度調整」：先前各期以「當期比例」暫估可扣抵的進項稅額，年度結束後須依全年實際免稅與應稅銷售額比例重新精算，多退少補，這也是403申報書「年底調整」欄位存在的原因（401申報書沒有這個欄位，因為401營業人沒有免稅收入需要按比例分攤）。

## 稅額計算欄：多出的特種稅額與國外勞務項目

403申報書稅額計算欄，除了401既有的銷項稅額、進項稅額、留抵稅額計算外，還多出：

```
一、購買國外勞務應納稅額：買受人依規定應繳納之營業稅（見「稅務相關法規」科目第2條第3款）
二、特種稅額計算之應納稅額：銷項特種稅額部分之應納稅額
三、中途歇業或年底調整補徵應繳稅額：兼營營業人中途歇業或年度結束調整時之應納（退）稅額
```

「本期應實繳稅額」的公式也隨之擴充為：銷項稅額合計＋購買國外勞務應納稅額＋特種稅額應納稅額＋年底調整應繳稅額之總和，再減除得扣抵進項稅額合計與上期留抵稅額。

## 常見錯誤

- 專營特種稅額計算業別（如專營銀行本業）誤用403申報，應改用404申報書
- 兼營營業人進項稅額直接全額扣抵，未依「應比例計算得扣抵進項稅額」欄規定按免稅銷售額比例分攤
- 忽略403申報書全年最後一期須做「年底調整」，未依全年實際比例重新精算暫估的進項稅額扣抵比例，導致多扣抵或少扣抵未被更正
