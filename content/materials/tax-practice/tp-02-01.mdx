## 兩大類課稅所得額計算方式

營利事業所得稅結算申報，依課稅所得額之計算方式，可分為兩大類：

**核實計算**：課稅所得額＝本年度營業收入總額－各項成本費用、損失及稅捐後之純益額（第24條，已於「稅務相關法規」科目介紹）。採核實計算者，依是否委託會計師查核簽證，分為「查帳申報」與「會計師簽證（稅簽）申報」。

**公式型所得**：課稅所得額以營業收入總額為基礎，乘上財政部核定的某個百分比算出，包括「擴大書審純益率」、「所得額標準」、「同業利潤標準」三種。

## 四種申報方式總覽

| 申報方式 | 全年所得額計算公式 | 適用對象 |
|----------|---------------------|----------|
| 擴大書審 | （營收淨額＋非營業收入）×擴大書審純益率 | 營收淨額及非營業收入合計3,000萬元以下（部分行業限1,000萬元以下）之中小企業，帳務不健全者 |
| 查帳申報 | 營收淨額－營業成本－營業費用＋業外收入－業外損失 | 帳證健全、收入費用明確者，不受營收多寡限制 |
| 會計師簽證 | 同查帳申報，經會計師查核簽證 | 全年營收及非營業收入達1億元以上、公開發行公司、金融業等依法應簽證者；其餘企業亦可自願採用 |
| 所得額標準／同業利潤標準 | 營收淨額×所得額標準（或同業利潤標準）＋業外收入－業外損失 | 帳務不健全、營收超過3,000萬元之企業；同業利潤標準另用於稽徵機關查核時未能提示帳簿憑證之情形 |

## 擴大書審計算範例

某公司從事零售業，行業代號對應之擴大書審純益率為6%。該公司全年營收淨額2,500萬元，非營業收入400萬元：

```
全年所得額 = （2,500萬＋400萬）× 6% = 174萬元
應納營利事業所得稅 = 174萬 × 20% = 34.8萬元
```

## 查帳申報計算範例

某公司全年營業收入4,000萬元，全年支出費用（成本＋費用）500萬元：

```
課稅所得額 = 4,000萬 － 500萬 = 3,500萬元
應納營利事業所得稅 = 3,500萬 × 20% = 700萬元
```

比較兩個範例可看出：查帳申報依實際盈虧課稅，若當年度實際純益率低於擴大書審純益率，查帳申報通常稅負較低；但查帳申報須負擔較高的記帳成本與被抽查風險。

## 所得額標準與同業利潤標準的差異

所得額標準（第80條）：稽徵機關依分業抽查核定各業所得額標準；納稅義務人申報之所得額若在標準以上，即以原申報額為準。

同業利潤標準（第83條）：納稅義務人於稽徵機關進行調查或復查時，未依限提示課稅所得額相關帳簿、文據者，稽徵機關得依查得資料或同業利潤標準核定所得額。同業利潤標準之淨利率通常高於所得額標準與擴大書審純益率，具有「未能舉證因而從高核定」的懲罰性質，實務上不建議企業一開始就以同業利潤標準規劃申報。

## 如何從收件編號判斷申報方式

結算申報書「收件編號」欄位第3碼即代表申報方式類別，常見代號：

```
1：會計師簽證申報案件
2：藍色申報案件
3：普通申報（收入淨額超過3,000萬元）案件
4：收入淨額3,000萬元以下查帳申報案件
7：3,000萬元以下擴大書面審核案件
```

記帳士協助客戶檢視歷年申報書時，可從此欄位快速掌握該公司過去採用的申報方式，作為評估是否變更申報方式的基礎。

## 常見錯誤

- 誤以為擴大書審一定比查帳申報省稅，忽略當年度虧損時擴大書審仍須按純益率繳稅，而查帳申報虧損年度可以不用繳稅
- 誤以為所得額標準與同業利潤標準是同一套數字，忽略同業利潤標準通常較高且具懲罰性質，是稽徵機關在企業未能提示帳證時採用的核定基準
- 誤以為申報方式一經選定就不能更改，忽略營利事業原則上可在每年度申報時依當年度帳務健全程度重新選擇最有利的申報方式
