# 記帳士歷屆試題查詢入口（常備參考）

供 §6.4 歷屆試題匯入與每年 12 月維運循環使用。若下列網址失效，改由 www.moex.gov.tw 首頁重新搜尋「考畢試題查詢」入口。

## 官方入口

- **考選部考畢試題查詢平臺**：https://wwwq.moex.gov.tw/
  - 依「考試年度」與「考試簡稱」篩選，記帳士考試屬「專技人員普通考試」類，簡稱含「記帳士」字樣
  - 試題檔案為 PDF 格式；公布時間為該次考試全部筆試結束之次日上午 10 時
  - 每年 11 月舉行的記帳士考試，試題通常於 11 月當月即可查得，12 月維運循環時應已可下載
- **考選部命題大綱查詢**：https://wwwc.moex.gov.tw/main/content/wfrmcontentlink4.aspx?inc_url=1&menu_id=154
  - 供每年 4–6 月查核當年度命題大綱使用（見 [annual-04-06-exam-syllabus.md](annual-04-06-exam-syllabus.md)）
- **考選部官網首頁**：https://www.moex.gov.tw/
  - 查詢當年度考試日期公告、應試須知

## 查詢步驟（考畢試題查詢平臺）

1. 開啟 https://wwwq.moex.gov.tw/
2. 「考試年度」選單選擇目標年度（民國年，如 115 = 西元 2026）
3. 「考試簡稱」選單搜尋含「記帳士」的考試名稱
4. 下載各科目試題與公告答案 PDF

## 版權注意

考選部試題為政府公開資訊，收錄至本站題庫時應標明出處年度與科目（`source: {"type": "past-exam", "year": <年度>, "number": <題號>}`），依 §6.4 規定辦理。

## 已知考試代碼（供快速重跑查詢）

- **114 年（西元 2025）記帳士考試**：考試簡稱代碼 `114180`（歸類於「114年專技人員高等考試建築師、25類科技師…暨普通考試不動產經紀人、記帳士考試」），類科代碼 `602`。各科目試題／答案下載網址格式：
  `https://wwwq.moex.gov.tw/exam/wHandExamQandA_File.ashx?t={T}&code=114180&c=602&s={S}&q=1`
  （`T`：`Q`=試題、`S`=答案、`M`=更正答案；`S`：`1301`=會計學概要、`1302`=記帳相關法規概要、`1303`=租稅申報實務、`1304`=稅務相關法規概要）
- **應試題型**（114 年簡章公告）：會計學概要為申論＋測驗混合題（各占 50 分）；租稅申報實務全為申論題；稅務相關法規概要、記帳相關法規概要為 50 題單選測驗題。租稅申報實務無官方測驗式標準答案檔（申論題人工閱卷）。
- **113 年（西元 2024）記帳士考試**：考試簡稱代碼 `113190`（「113年專技人員高等考試建築師、32類科技師…暨普通考試不動產經紀人、記帳士考試」），類科代碼 `602`。科目代碼與 114 年不同：`1401`=會計學概要、`1402`=記帳相關法規概要、`1403`=租稅申報實務（申論，無測驗答案檔）、`1404`=稅務相關法規概要。會計學概要有更正答案檔（`t=M`）：第8題答A或C均給分、第13題答A或B均給分。
- **科目代碼逐年不同**：不可沿用前一年代碼。查法：開啟查詢平臺選定年度與考試簡稱後按查詢，於結果頁以 JavaScript 擷取「專技普考_記帳士」區塊內所有 `wHandExamQandA_File.ashx` 連結即得當年度 `s` 代碼。
- **送分題（均給分）處理慣例**：計分引擎為單選精確比對，`answer` 只存一個主答案（依台灣法規實務較正確者），詳解中註明官方更正公告與另一可接受選項，並加 `送分題` tag。
- **PDF 轉文字注意**：firecrawl 的 PDF 解析在此平台的試題檔上，約 10% 題目因排版分頁會遺失選項文字；改用 Bash 下載 PDF 後以 Read 工具直接讀取（多模態視覺解析）可完整還原，建議優先採用此法。
