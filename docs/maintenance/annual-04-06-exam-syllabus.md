# 每年 4–6 月：考選部命題大綱查核

對應 [DEVELOPMENT_PLAN.md](../../DEVELOPMENT_PLAN.md) §8.1、§5。目標：確認當年度記帳士考試科目與命題大綱是否異動，若有則同步調整本站科目地圖。

## Checklist

- [ ] 1. 以 WebSearch 至考選部官網（www.moex.gov.tw）查詢當年度記帳士（專技普考）考試公告：考試日期、應試科目名稱
- [ ] 2. 至考選部「命題大綱」頁面（wwwc.moex.gov.tw，路徑可能隨改版變動，找不到時從 www.moex.gov.tw 首頁的「應試須知」或「命題大綱」入口重新搜尋）查詢記帳士各專業科目最新命題大綱
- [ ] 3. 逐科比對本站 [DEVELOPMENT_PLAN.md](../../DEVELOPMENT_PLAN.md) §5 科目地圖（會計學、租稅申報實務、稅務相關法規、記帳相關法規）與最新命題大綱：
  - 科目名稱是否變更
  - 章節範圍是否增刪（例如新增法規、廢止法規）
  - 是否新增應試科目
- [ ] 4. 若有差異，先更新 §5 科目地圖與文末變更紀錄，說明變更理由與依據的官方公告，再排入下次內容補完或修訂工作
- [ ] 5. 若無差異，仍在對應 GitHub Issue 留言記錄「本次查核無異動」與查核日期，作為下次接手者的追蹤依據

## 完成標準

- [ ] 已查證當年度考科與命題大綱
- [ ] 若有異動，§5 科目地圖與變更紀錄已同步更新；若無異動，已留言記錄查核結果
